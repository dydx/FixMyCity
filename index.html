Fix My City

This is my submission to the 2015 National Day of Civic Hacking. Fix My City aims to be a comprehensive web based service to empower citizens to inform their maintenance and utilities departments of things that need to be fixed.. in their city.

dashboard

It is still a work in progress. Currently the administrative console is about 70% complete, with a lot of work needed still on the client-facing side of the application.

In a nutshell, the client-facing bit of Fix My City allows citizens to geotag areas of interest along with a short description of the problem at hand. That information is made immediately available to the administrative console as a grid of map locations with addresses and descriptions, color highlighted by severity.

To install Fix My City, the following requirements need to be met:

  • PHP 5.6+ (w/mcrypt extension)
  • MySQL
  • Composer
  • Bower

To get the code and install it, log into your environment or server and run:

git clone https://github.com/dydx/FiMyCity

then, move that directory into your web root or just in your local build path. Run bower install and composer install, and let the required packages be downloaded.

To migrate the database and get setup, you will need to set the correct MySQL connection variables in the .env file, then run the following command in the root of the FixMyCity directory:

php artisan migrate

This will set up your database tables.

You may then register a new user account and start using Fix My City.
